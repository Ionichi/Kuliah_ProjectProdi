Cara Menjalankan Project:
1. Setelah melakukan clone project, buka folder project tersebut dan buka ProjectProdi.sln
2. Jalankan Build terlebih dahulu (Shortcut: CTRL + B) untuk membuild .exe dari project.
3. Sesuaikan Database dengan database lokal masing-masing. (Note: Di file ConnectionModule.vb
4. Selesai, aplikasi dapat berjalan dengan baik.
